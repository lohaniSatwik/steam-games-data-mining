## Project Overview

This project applies data mining and machine learning to a dataset of ~122,000 Steam games. The goal is to **predict whether a game will receive a positive or negative review outcome** (binary classification: Good / Bad) based on its observable metadata attributes such as genre, price, tags, and platform availability.

Beyond prediction, we use SHAP values and feature importances to identify which game attributes most strongly drive positive reception.

## Data Description

| File | Description |
|------|-------------|
| `X_classification.parquet` | 151 features: log-price, genre/category/tag dummies, platform flags, playtime, era buckets |
| `y_classification.parquet` | Binary label: 1 = Good (≥70% positive reviews), 0 = Bad |
| `continuous_cols.json` | Column names to apply `StandardScaler` to inside CV pipelines |

**Important:** Features are saved **unscaled**. Each modelling notebook must apply `StandardScaler` inside its `sklearn.Pipeline` so that scaling is fit only on training folds, not on the full dataset.

## Modelling Notes

- **Evaluation metric:** Macro F1-score (primary), AUC-ROC and Precision-Recall AUC (secondary)
- **Cross-validation:** 10-fold stratified CV (outer loop); 3-fold inner loop for hyperparameter tuning
- **Class imbalance:** 71.4% Good / 28.6% Bad — use `class_weight='balanced'` and SMOTE
- **Scaling:** Use `sklearn.Pipeline` with `StandardScaler` inside each model notebook. The columns to scale are listed in `continuous_cols.json`

Example pipeline setup:
```python
import json
import pandas as pd
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LogisticRegression

X = pd.read_parquet('Data/processed/X_classification.parquet')
y = pd.read_parquet('Data/processed/y_classification.parquet')['label']

with open('Data/processed/continuous_cols.json') as f:
    continuous_cols = json.load(f)

# Scale only continuous columns inside the pipeline
from sklearn.compose import ColumnTransformer
preprocessor = ColumnTransformer([
    ('scaler', StandardScaler(), continuous_cols)
], remainder='passthrough')

pipeline = Pipeline([
    ('preprocessor', preprocessor),
    ('classifier', LogisticRegression(class_weight='balanced', max_iter=1000))
])
```
